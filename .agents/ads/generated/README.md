# `.agents/ads/generated/` — Agent Output

This folder is where the Meta ads agent drops outputs:
- New static image ads (`.png`, `.jpg`)
- Generated ad copy briefs (`.md`)
- Campaign launch specs (`.json`) before they're pushed to Meta
- Weekly performance reports (`.md`)

**Convention:** Filenames are dated and angle-tagged so the history is browsable.

Examples:
- `2026-05-18_angle-guarantee_static-01.png`
- `2026-05-18_copy-batch.md`
- `2026-05-20_launch-spec_campaign-01.json`
- `2026-05-22_weekly-report.md`

Old outputs aren't deleted — they're the historical record of what we've tried and what worked.
