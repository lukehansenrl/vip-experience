# Thumbnails Library — The Visual Source-of-Truth

**Replaces all AI image generation in our ad workflow.** We do not use Higgsfield, GPT image 2, Midjourney, or any other image generator. We use existing professional thumbnails from the SpookLuke library — designed by real thumbnail artists, proven across years of YouTube performance, instantly recognizable to the Rocket League audience, and stylistically locked to the visual language RL viewers already trust.

**File types:** `.jpg`, `.png`, `.webp` — all 1920×1080 YouTube-thumbnail-spec. Need to be re-exported to ad placements:
- **1:1 square** (1080×1080) for Meta feed
- **9:16 vertical** (1080×1920) for Reels / Stories

The original 16:9 thumbnails can be cropped to 1:1 or used as-is in 1.91:1 placements. For 9:16, the agent will spec a re-crop with the car/rank-emblem as the focal element and burn the copy vertically.

## Workflow for using the library

The agent's job, weekly:

1. **Read this file** — start with the curated picks above.
2. **Pull the actual image** from the library path.
3. **Re-export to ad placements** (1:1 square + 9:16 vertical) — agent specs this as an ffmpeg / ImageMagick command if needed, or hands off to Luke for manual export.
4. **Burn in the suggested copy** (see the per-row "Copy burn-in" column). Use the burn-in style guide below.
5. **Pair with primary text + headline** generated separately.
6. **Drop final assets in `.agents/ads/generated/YYYY-MM-DD_angle-X/`** with the source-thumb filename in the output filename so we can trace what worked.
7. **Re-curate quarterly** — as new thumbnails get added in `2026/March/`, `2026/April/`, etc., re-run the angle-fit pass.

### Burn-in style guide

When adding text overlay to a source thumbnail for ad use:

- **Font:** Bold sans-serif (Inter Black, Anton, or whatever the source thumb already uses)
- **Size:** Big — text should be readable at thumb-in-feed scale (~300px wide)
- **Color:** White or yellow with thick black/dark outline OR drop shadow for legibility against any background
- **Position:** Bottom-third or wherever it doesn't obscure the rank emblem
- **Length:** 2-5 words max. The hook is in the headline/primary text; the burn-in is the scroll-stopper.

**Never burn in:**
- The price ($497)
- A URL (the ad already has a link)
- The phrase "Apply now" (let the headline carry the CTA)
- More than one line of text (one line, one message)
